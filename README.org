* Status
The eAppendix for SAS macro for causal mediation analysis with survival data (Valeri & VanderWeele 2015) describes the following grid of models.

| yreg \ mreg     | linear | logistic |
|-----------------+--------+----------|
| linear          |        |          |
| logistic        |        |          |
| loglinear       |        |          |
| poisson         |        |          |
| negbin          |        |          |
| survCox         |        |          |
| survAFT exp     |      1 |        1 |
| survAFT weibull |      1 |        1 |
